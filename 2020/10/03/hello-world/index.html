Ubuntu 環境安裝 hexo

  1. 安裝 npm: sudo apt-get install npm

  2. 安裝 git: sudo apt-get install git

  3. 安裝 hexo: npm install -g hexo-cli

  4. 設定 hexo

    • hexo init myblog
    • cd myblog; npm install
    • vim _config.yml
      • title: 我的標題
      • subtitle: 我的副標題
      • description: 我的說明
      • author: My name
      • language: zh-TW
    • 執行 hexo: hexo server
    • 執行瀏覽器: http://localhost:4000
    • 變換 Hexo 主題 (theme)
      • Hexo 預設主題為 landscape，更換為 Next
      • git clone https://github.com/theme-next/hexo-theme-next themes/next
      • vim _config.yml
        • 將原來 theme: landscape 變更為 theme: next
        • 重新啟動 hexo: hexo server
    • Next 主題設定
      • scheme
        • Next 有不同的 schemem，預設為 Muse
        • vim themes/next/_config.yml
        • 將原來 scheme: Muse 變更為 scheme: Pisces
        • 重整 http://localhost:400
      • 代碼區塊
        • Next 預設代碼區塊為白底
        • 將原來 highlight_theme: normal 變更為 hightlight_theme: night
      • 選單設定
        • 在 menu: 區塊可設定相關選單的顯示
          • 移除 about 前的 #(註解)，可在選單顯示 關於
            • Hexo 產生 關於
              • hexo new page "about"
              • 編輯 source/about/index.md
          • 移除 tags 前的 #，可在選單顯示 標籤
            • Hexo 產生 標籤
              • hexo new page "tags"
              • vim source/tags/index.md
              • 給文章添加 tags: 屬性
            • 若要刪除之前的 tag，先清除 hexo 後，在重新啟動
              • hexo clean
              • hexo server
      • 開啟社群帳號連結
        • 在 social: 可設定社群帳號連結
          • GitHub: https://github.com/your-user-name

  5. 佈署至 github

    • github 設定
      • 登入 github
      • New repository
        • Repository name: <your name>.github.io
        • Public
        • Create repository
      • <your name>.github.io -> Settings
        • Choose a theme: Any theme you want
      • https://<your name>.github.com
    • 安裝 npm 套件: npm install hexo-deployer-git --save
    • 編輯 _config.yml
    • hexo deploy
    • 更新 github: hexo clean && hexo generate && hexo deploy

  6. misc

    • scaffolds 目錄下是各種模板，執行新建命令時會根據這裡的模板作為預設值，所以可依據自己的需求修改相關模板變更預設值。例如，修改 post.md 來建立 post 的預設值：
      • hexo new “mypost”
    • 使用 hexo-admin 後台管理工具
      • npm install hexo-admin –save
      • hexo server
      • http://localhost:4000/admin
